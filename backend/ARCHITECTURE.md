|                                        TOFIMOTIA BACKEND SYSTEM                                            |
+----------------------------------------------------------------------------------------------------------+

        +-----------|-----------+      +-----------|-----------+     +------------|-----------+
        |           |           |      |           |           |     |            |           |
+-------v---+ +-----v----+ +----v----+ +-----v----+ +----v----+ +---v----+ +----v----+ +----v----+
